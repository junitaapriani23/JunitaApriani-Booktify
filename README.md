# Booktify
Booktify merupakan aplikasi untuk membaca dengan mudah tanpa kesulitan ke perpustakaan atau membawa buku cukup dengan gadget maupun elektronik lainnya 

## Fitur Utama
- Membaca dengan mudah
- Pembayaran simple
- Dimanapun dan kapanpun
